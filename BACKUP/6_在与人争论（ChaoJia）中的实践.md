# [在与人争论（ChaoJia）中的实践](https://github.com/xushulin/blog-S.L.Xu/issues/6)

首先，自己要是正义的，只有正义的一方才会获取最终的胜利。

# 抓住主动权

争论中的主动权，就是提问权。

- 提问就是进攻，如果一味的只是回答对方的提问，则永远不可能赢，回答总会有漏洞并且会受到对方的连续追问。

- 没有充分的准备不要回答，或者以提问的方式进行回答。保证自己的攻势，从而保证主动权。

- 在部分情况下，可以忽略对方的进攻。例如有些进攻往往是离谱的，不可以理解的；有些进攻是疯狂的没有思路的，基于歪理的。对方的类似这种进攻，完全可以忽略。他打他的，我打我的。打得过就打，打不过就跑。


# 抓住主要矛盾

- 抓住敌人的薄弱点，进行追问

- 抓住争论的重点，进行追问


# 依靠人民群众

因为自己是正义的，或者说自己所做的事业是正义的，所以完全不要害怕身边的群众，反而还要依靠群众，团结群众，群众的力量是无比强大的。寻找群众的帮助。


# 不为外部干扰的坚定决心

- 保持定力，内部的稳定是取胜的必要条件。如果自己破防了，则一定会失败。
